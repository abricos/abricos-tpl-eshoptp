English:
Do not edit this pattern, since it comes with the distribution.
Therefore, in the case of updating the platform, all of the changes in this pattern are
canceled.

To make changes to a template, copy it to another template (with a new name)
then go into the control panel and in the "Site Settings" -> "Style"
Select a new template.

Русский:
Не редактируйте этот шаблон, так как он поставляется с дистрибутивом.
Поэтому в случае обновление платформы, все изменения в этом шаблоне будут
отменены.

Для внесения изменений в шаблон, скопируйте его в другой шаблон (с новым названием), 
потом зайдите в панель управления и в разделе "Настройки сайта" -> "Стиль" 
выберите новый шаблон.
